# Chạy thuật toán

Để chạy thuật toán, đầu tiên truy cập vào thư mục chứa file mã nguồn tương ứng. Tại đó gõ lệnh

 `python tên_file.py`

 Ví dụ để chạy thuật toán C45:
 ---

 `>` pwd

 */doan3*

 `>` cd c45

 `>` python main.py
